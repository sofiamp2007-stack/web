Comandos GIT

  • git init- criar um repertório;
  • git add[file] :adiciona um arquivo no stage
  • git status: exibe o estado do repositório;
  • git commit: -m "msg" : cria um commit
  • git log : exibe o ccomits de repertório, da branch atual
  • git checkout referencia: Recupera o conteudo da referencia para WK
  • git remote add nome url : definir um repositorio remoto
  • git push url branch: envia o branch para o repositorio remoto
